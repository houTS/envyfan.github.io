肥皂君Android之路–自我介绍

发表于 2016-03-09   |   分类于 肥皂君Android之路   |  

我叫Vv,因为名字中有个“威”字，小时候小伙伴们喜欢叫我“威威”（女生居多^_^），后来，随着样子不断越长越残，为了叫起来不恶心慢慢地就演变成Vv（哈哈哈哈）。至于为什么这系列会叫肥皂君呢= -，女票说我像肥皂，那肥皂就肥皂吧= -。

我是2013年毕业的专科生，专业是软件开发，在校主修JavaWeb开发选修Android开发，还有其他乱七八糟的什么数据结构、计算机原理、软件工程、网页设计、数据库、动画赏析玛雅之类的，唔~至于学到什么程度嘛，实在惭愧得不得了，完全愧对于2W一年的学费T_T。我特么那时连Java里很基础的什么是内部类、泛型都不知道，只知道跟着教材写。交大作业的时候参考教科书用SSH弄个会员管理系统就以为自己很牛X= -，到期末大作业展示时，看到别人的系统简直就是要UI有UI要功能有功能完全不逊色于一些大网站，想想自己照着书参考出来的会员管理系统是多么的2B。现在想起当初的自己，也是醉了（哈哈）。

上课之余，别的时间我都花在了和舍友打LOL和社团活动里，认识了一帮好玩又好人的师兄，跟着他们到别的学校表演BeatBox，还认识了现在的女票。虽然学业上算是完全荒废的，但是在经历上也算是过得精彩，整个大学时光无悔啦。0 0

临近毕业我好好地反省了一下自身情况，觉得在大学的三年里，自己并没有把心思放在为以后找工作上，按照当前的水平出来肯定很难找到合自己心意的工作，倒不如插本弄个更好的文凭，并在学校好好地修炼修炼自己到时再出来找，可惜偏偏命运喜欢捉弄人，尽管我是选择了插回本校把插本难度降到最低（政治数英平均40就完事了）。最终还是以119差一分与本科无缘，我真特么是个文盲= -。懵了好几天不知道自己下一步该怎么办，在机缘巧合之下给同学介绍了我去人生中的第一家IT公司工作，从此开始了我的Android程序员生涯。
